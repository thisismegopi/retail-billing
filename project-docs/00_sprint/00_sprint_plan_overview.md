## Key Milestones

### Milestone 1: MVP (End of Sprint 2)

- Functional Authentication.
- Ability to manage products.
- Ability to generate bills and print receipts.
- _Goal: Can use the software for basic "Cash & Carry" sales._

### Milestone 2: Full Sales Cycle (End of Sprint 3)

- Inventory automatically tracks stock.
- Credit customers can be managed.
- Payments can be collected later.
- _Goal: Can replace existing ledger books._

### Milestone 3: Production Ready (End of Sprint 6)

- Full reporting suite.
- Mobile PWA readiness.
- Security rules enforced.
- _Goal: Full deployment to shop floor._
